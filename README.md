## Summary
A full-stack food ordering web application built with Spring Boot and React, featuring secure authentication, online payments using Stripe, and AWS S3 image storage.

## Skills and Technologies Used
### Programming

- **Languages**: Java, Javascript
- **Frameworks**: SpringBoot, Spring Data JPA, Spring Security, React, Axios, Stripe API, AWS S3
- **Database**: MySQL
